Scratch Programming Intro

Grades 3-6
Beginner to Intermediate
The perfect first course in programming! Scratch is a visual language designed for kids by the MIT Media Lab. It removes early challenges to coding such as syntax errors and typing skills by using a visual, drag and drop interface.
